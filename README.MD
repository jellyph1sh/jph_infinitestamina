# JPH_INFINITESTAMINA

## DESCRIPTION:
Simple script to have infinite endurance stamina.

## HOW TO INSTALL:
- Download the script with `git clone https://github.com/jellyph1sh/jph_infinitestamina.git` or click on `Download ZIP`.
- Put the folder in your `resources` folder.
- Add `ensure jph_infinitestamina` in your `server.cfg` file.
- Now, just run your server!

## FEATURES:
- Infinite stamina loaded on script starting.
- Infinite stamina unloaded on script stopping.
- Message in the server console when starting and stopping the script.
